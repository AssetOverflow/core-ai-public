# Public Benchmark Dashboard

This directory contains sanitized public benchmark summaries.

The purpose of these summaries is to:

- demonstrate measurable progress
- communicate evaluation rigor
- provide transparent limitations
- preserve benchmark integrity
- avoid implementation leakage

## Planned Public Benchmark Categories

Potential public benchmark categories include:

- repeatability
- semantic drift
- hallucination stress tests
- retention
- transfer
- memory efficiency
- long-context behavior
- quality-per-compute

## Current Status

Current files are scaffolds only.

Future updates may include:

- CSV summaries
- JSON summaries
- charts
- trend lines
- public comparison reports
- sanitized evaluation examples

## Important Boundary

This directory intentionally excludes:

- benchmark generators
- hidden prompts
- private datasets
- scoring internals
- implementation details
- architectural mechanisms
