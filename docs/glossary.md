# Public Glossary

This glossary defines public-facing CORE terminology.

The definitions here are intentionally high level and non-enabling.

## Hallucination

A model output that presents unsupported, fabricated, or unjustified information as if it were true.

## Reliability

The degree to which a model produces accurate, stable, and trustworthy outputs across repeated evaluations.

## Repeatability

The ability of a system to produce meaningfully consistent outputs under equivalent conditions.

## Deterministic Behavior

Behavior where equivalent inputs and conditions produce equivalent outcomes.

Public discussions of determinism in this repository refer to reliability and output stability goals, not guarantees of perfect determinism in all settings.

## Structured Mastery

The ability to reuse learned structure across related problems rather than relying on shallow pattern matching.

## Transfer

The ability of a model to apply previously learned knowledge to new but structurally related tasks.

## Retention

The ability of a model or system to preserve and reuse learned information over time or across boundaries.

## Context Window

The amount of information directly available to a model during inference.

## Memory Efficiency

The relationship between capability, memory usage, and compute cost.

## Semantic Drift

Meaningful deviation in outputs across repeated or extended generations.

## Negative Control

An evaluation condition that should not improve if a claimed capability is genuine.

## Evaluation Hardening

The process of strengthening evaluations against leakage, shallow shortcuts, benchmark overfitting, or misleading metrics.

## Benchmark Integrity

The principle that benchmarks should measure genuine capability rather than memorization, leakage, or optimization against the test itself.

## Public Summary

A sanitized description of results or behavior that avoids exposing proprietary implementation details.

## Implementation-Level Detail

Specific technical information that could materially help reproduce CORE internals.
